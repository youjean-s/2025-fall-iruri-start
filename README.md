# Team 이루리 | AI 기반 청년 금융 케어 서비스: FINNUT

이 저장소는 이화여자대학교 졸업프로젝트 스타트 수업 (2025 Fall)을 위한
**팀 이루리(Iruri)**의 **초기 기획 및 핵심 기술 검증용** 레포지토리입니다.

FINNUT은 청년층의 지출/저축 패턴을 AI로 분석해, 맞춤 재정 관리 계획을 제안하고 올바른 금융 습관 형성을 돕는 청년 스마트 금융 케어 서비스입니다. 


---

## 🚀 프로젝트 개요 (FINNUT | Version 1.0)

| 구분 | 내용 | 상세 설명 |
| :--- | :--- | :--- |
| **프로젝트명** | **AI 기반 청년 금융건강지수(FHI) 분석 및 맞춤 복지/장학금 자동 매칭 케어 서비스: FINNUT** | 
| **주제** |**청년층의 재정적 불안정 해소 및 금융 습관 개선** | 
| **핵심 목표** | 1. **AI 기반 FHI 엔진**을 통한 위험 소비 패턴 진단 및 코칭. 2. **API 및 웹 크롤링**을 통한 흩어진 청년 복지/장학금 정보 자동 매칭. |
| **상태** | **핵심 로직 구현 및 기술 검증 완료** (Rule-based FHI, 정책 데이터 수집/매칭) 최종 개발 단계 진입을 위한 기반 마련. |

---

## ⚙️ 주요 기술 스택 및 활용

| 영역 | 주요 기술/패키지 | 활용 목적 |
| :--- | :--- | :--- |
| **AI/ML** |**LightGBM**  | FHI 하락 예측 및 이상 소비 패턴 탐지를 위한 ML 모델 학습 (확장 계획). |
| **코칭/NLG** |**ChatGPT API**  | FHI 진단 결과 기반, 사용자 맞춤형 3분 코칭 카드 생성. |
| **데이터 처리** |**Python (re, datetime)** | 카드 알림 텍스트 파싱 및 FHI 지표 계산. |
| **정보 수집** |**공공데이터 API (한국장학재단 등)**& **웹 크롤러**  | 장학금 및 청년 복지/금융사 혜택 정보 통합 확보. |
| **모바일 DB** |**Room DB** (온디바이스 DB)  | 개인 금융 데이터의 **로컬 저장 및 분석**을 통한 프라이버시 보호. |
